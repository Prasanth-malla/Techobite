# Techobite

**Techobite** is a construction website platform that allows users to create and manage professional websites tailored to specific business needs. It provides intuitive tools for building responsive and visually appealing websites, ideal for industries like construction.  

---

## Features

- **Responsive Design:** Works seamlessly on desktops, tablets, and mobile devices.  
- **Service Pages:** Showcase all services with detailed descriptions.  
- **Portfolio:** Display completed projects with filtering and lightbox functionality.  
- **Blog Section:** Share news, updates, and articles.  
- **User Management:** Manage admin and client accounts efficiently.  
- **Smooth Navigation:** Scroll-to-section with active state highlights.  
- **Mobile-Friendly Navigation:** Hamburger menu and dropdowns for mobile screens.  
- **Back-to-Top Button:** Easy scrolling for long pages.  
- **Interactive Sliders:** Clients and portfolio sliders powered by Swiper.js.  
- **Filterable Portfolio:** Isotope.js used for filtering portfolio items dynamically.  
- **Lightbox Preview:** GLightbox for portfolio images and project details.

---

## JavaScript Functionality

The main JS (`main.js`) handles:

- Navbar active link highlighting on scroll.
- Smooth scrolling to sections.
- Fixed header on scroll with offset.
- Mobile navigation toggling and dropdown activation.
- Back-to-top button visibility.
- Clients and portfolio sliders (Swiper.js).
- Portfolio filtering (Isotope.js).
- Portfolio lightbox functionality (GLightbox).
